# mongo-cmd
cmd in mongo


#### show dbs

查看所有的数据库

#### use test

设置当前的数据库

#### db

查看当前连接的数据库名称

#### show collections

查看当前数据库所有的 collection

#### db.printCollectionStats()

查看当前数据库所有的 collection 的状态



#### exit

退出 shell
